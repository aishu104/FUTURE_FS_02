- Login system (LocalStorage-based)
- Protected routes
- Auto session handling

### 📊 Dashboard
- Leads count
- Contacts count
- Deals count
- Tasks count
- Real-time updates

- Contacts (CRUD + Search)
- Deals Pipeline (Stages: New, In Progress, Won, Lost)

- Reports (Chart.js analytics)
